This project will include the following:

- Visual GUI "To-Do List" that will have the following panels:
    1. Panel to add new items to your list
    2. Panel to remove from your list if the list is not empty
    3. colorize the background of an item
    4. modify an entry
    5. sort by date or name option (will be added in future updates)

- an App that is made to autostart whenever the pc turns on (first version will be Windows only!)

- a main panel that will be made with tkinter, and it will be scalable in future updates
